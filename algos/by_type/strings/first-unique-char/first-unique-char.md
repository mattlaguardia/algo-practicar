# String - First Unique Character

#### Prompt:

Given a string, find the first non-repeating character in it and return it's index. If it doesn't exist, return -1.

#### Example:

```
s = "pomeranian",
return 0.

s = "lululemon",
return 5.

s = 'lukeluke',
return -1.
```

#### Note

Assume the string only contains lowercase letters.
